# Fintrack

Fintrack adalah aplikasi manajemen keuangan berbasis Laravel.

## Cara Menjalankan
1. Clone repository
   git clone https://github.com/niaanggraini/fintrack.git
2. Masuk folder project
   cd fintrack
3. Install dependency
   composer install
4. Copy file environment
   cp .env.example .env
5. Generate key
   php artisan key:generate
6. Jalankan server
   php artisan serve
